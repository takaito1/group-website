Description

This is an attempt to compress the data volume of earth system model output in order to increase accessibility. The original model output for ocean variables is reduced to annual and global 2x2 degree resolution with 33 vertical z-levels. Historic and RCP8.5 scenario segments are combined to a single time series consisting 1920 to 2100. The piContorl fields are extracted from Year 400 (equivalent of 1848) to 1399 at the same resolution. The ensemble member #1 is initialized with the Year 402 of the piControl (i.e. Jan 1st, 1850). Thus Year 472 is the equivalent of 1920 (which is the first year of the ensemble runs). This can be used to correct for the model drifts.

Disclaimer. This is a research product that is not an official part of the CESM-LENS project. If you find somehting wrong, please send information to taka.ito (at) eas.gatech.edu

The files are named as: ENS_MODEL_RESOLUTION_FORCING_YEAR1_YEAR2.nc. Each file contains 4D (lon,lat,dep,time) fields of potential temperature, salinity, potential density, oxygen, phosphate, dic, alk, ideal age, OUR, and 3D (lon,lat,time) fields of air-sea CO2 flux, air-sea O2 flux, air-sea heat flux, air-sea freshwater flux, fractional ice cover, and maximum mixed layer depth (annual max).

ENS: ensemble member number

MODEL: model name

RESOLUTION: 2x2 (horizontal) L33 (vertical 33 levels) ym (yearly mean)

FORCING: hcp85 (historic+rcp85)
